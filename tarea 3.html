● ¿Qué problema resuelven las herramientas virtualenv y RVM (o rbenv)? ¿Para qué
lenguaje se utiliza cada una? si no utilizas python o ruby, ¿cuál sería la alternativa para tu

Las herramientas Virtualenv y RVM se utilizan para resolver el problema de usar varias
herramientas para hacer testing, desarrollo de programas y hasta desarrollo web. Ambas
se utilizan para agrupar en un mismo entorno varias herramientas utiles.
VirtualEnv se utliza para el lenguaje Python.
RVM se utiliza en Ruby.
